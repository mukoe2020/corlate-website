# Corlate Technologies

Professional landing page for Corlate Technologies — a digital marketing and web development agency based in South Africa.

## Features

- 🎨 Modern, responsive design with dark theme
- ✨ Smooth animations and interactions
- 📱 Mobile-friendly (responsive breakpoints)
- 🚀 Fast performance optimized
- 🔍 SEO-friendly structure
- 💬 WhatsApp integration
- 📋 Contact form with validation feedback
- 📊 Services showcase
- 👥 Team section
- 💬 Client testimonials
- 🔄 Process workflow
- 🏆 Trust/Stats section

## Getting Started

1. Clone or download this repository
2. Run the web server:
   ```bash
   npm start
   # or
   python3 -m http.server 8000
   ```
3. Open http://localhost:8000 in your browser

## Customization

### Update Contact Information
- Replace `YOURNUMBER` with actual WhatsApp number
- Update email: `info@corlate.co.za`
- Update location: Johannesburg, South Africa

### Add Media Assets
- Place team photos in `assets/images/team.jpg`
- Place logo in `assets/images/logo.jpg`
- Place hero video in `assets/videos/hero.mp4`

### Colors
Edit CSS variables in `<style>` section of `index.html`:
```css
:root {
  --teal: #0d6e78;
  --gold: #f5b319;
  --sky: #5bd4e0;
  /* ... */
}
```
